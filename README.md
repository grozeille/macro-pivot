# Macro Pivot

## Prepare

UI

```
npm install
```

Python

```
cd python
py -m virtualenv venv
.\venv\Scripts\activate.bat
pip install -r requirements.txt
```

## Build / Test

```
npm start
```

## Package

```
npm run dist
```
